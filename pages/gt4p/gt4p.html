Welcome to Gran Turismo 4 Poundland Edition.


This project sets the price of all cars to 1CR and allows you to purchase special colors without needing to unlock them first.

How to Patch

Note: You must use Gran Turismo 4 Online Public Beta. Other versions will not work.

  1. Download an xDelta patcher (such as xdeltaUI or xdelta3).
  2. Run the patcher and select your Gran Turismo 4 Online Public Beta disc image as the source file. (Make sure to make a backup copy first!)
  3. Select the provided .xdelta patch file.
  4. Choose an output location and filename for the patched game.
  5. Start the patching process. This may take a few moments.
  6. Once complete, a new patched copy of the game will be created.
  7. Load the patched game. If successful, the mod will now be applied.
